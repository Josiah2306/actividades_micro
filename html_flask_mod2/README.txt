Aplicación de Plantillas con Flask - Ejemplo Básico

Esta es una aplicación sencilla desarrollada con Flask que utiliza plantillas HTML dinámicas mediante Jinja2.

Páginas incluidas:
- / : Página principal con un mensaje de bienvenida.
- /usuarios : Muestra una lista de usuarios provenientes del servidor (back-end).

Características:
- Uso de una plantilla base (base.html) para mantener un diseño consistente.
- Visualización de contenido dinámico basado en datos del servidor.
- Separación clara entre la capa de presentación (Front-End) y la lógica de negocio (Back-End).

Cómo ejecutar la aplicación:
1. Asegúrese de tener Flask instalado.
2. Ejecute el archivo: `python app.py`
3. Acceda a la aplicación desde el navegador en: http://localhost:5000
