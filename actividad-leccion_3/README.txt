# Sistema de Gestión de Usuarios

Este es un proyecto sencillo realizado en Flask que proporciona un API para gestionar usuarios. El servidor tiene tres rutas principales:

1. **GET /info**: Devuelve información general sobre la aplicación en formato JSON.
2. **POST /crear_usuario**: Recibe los datos de un usuario (nombre y correo) y valida que ambos estén presentes antes de añadirlo.
3. **GET /usuarios**: Devuelve una lista de los usuarios almacenados en memoria (diccionario o lista).

## Requisitos

- Python 3.x
- Flask
