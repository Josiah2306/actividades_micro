Capstone Demo App

Este proyecto es una demostración simple de una arquitectura Cliente/Servidor usando Flask como backend. Fue creado como parte de una actividad práctica para entender cómo funcionan las rutas REST y cómo interactúan los distintos componentes del sistema.

Rutas disponibles:

    1. GET /info

      Devuelve información básica sobre la aplicación (nombre, versión y estado).

    2. POST /mensaje

      Recibe un mensaje en formato JSON y devuelve una respuesta personalizada confirmando la recepción.

Cómo ejecutar:

Instala Flask si no lo tienes:

    pip install flask

Ejecuta el servidor con:

    flask run

Usa el REST Client en Visual Studio Code para hacer solicitudes a las rutas disponibles.

Requisitos:

1. Python 3
2. Flask
3. Visual Studio Code con REST Client
